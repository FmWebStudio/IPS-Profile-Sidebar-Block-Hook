Описание

Добавляет боковой блок с информацией о профиле.

Шаг 1: Залейте файлы хука

Войдите в FTP и залейте содержимое папки upload в корень папки с вашим стилем.

Шаг 2: Импортируйте файл хука

Войдите в АЦ и перейдите к Система -> Приложения -> Управление хуками . Нажмите на кнопку "Установить хук", и выберите XML файл с архива и нажмите Установить.
